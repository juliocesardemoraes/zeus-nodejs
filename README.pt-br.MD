## Primeiros passos

#### Como rodar esse projeto?

1- Tenha o [Node.js](https://nodejs.org/en) instalado. E o [git](https://git-scm.com/) deve também deve estar instalado na sua máquina.

2- Rode o comando

```
npx zeus-nodejs pt
```
